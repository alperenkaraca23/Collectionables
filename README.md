# Collectionables

Collectionables is an app that allows users to create and manage collections, where they can add items to their collections with photos and notes. This app is developed using the MVVM architecture and SwiftUI for the user interface.

## Features

- Users can create collections and give them a name.
- Each collection can have items added by the user, which include photos from their gallery and notes.
- Users can easily edit, view, and update their collections.
- The app provides a rich experience with different views: `AddCollectionView`, `AddItemView`, `CollectionDetailView`, and `ContentView`.

## Technologies Used

- **SwiftUI**: Used for creating the user interface.
- **MVVM**: The MVVM architecture is implemented to make the code more modular and maintainable.

## App Flow

1. **Create Collection**: When users open the app, they can create a new collection and give it a name.
2. **Add Collection Item**: Users can add items to their created collections. Each item can be saved with a photo and notes.
3. **Collection Details**: Each collection item can be viewed and edited in detail.

## View Structures

- **AddCollectionView**: The screen for creating a new collection.
- **AddItemView**: The screen for adding items to a collection.
- **CollectionDetailView**: The screen for viewing and editing the details of a collection item.
- **ContentView**: The main screen where users can view their collections.
